## 📝 Quick Test Steps

1. **Open Login Page**
   - See split-screen with D-Mart illustration
   - Enter 10-digit mobile number
   - Click "CONTINUE"

2. **OTP Input Page**
   - See phone icon (desktop)
   - Enter mobile number again
   - Click "SEND OTP"
   - See success message

3. **OTP Verify Page**
   - See shield icon (desktop)
   - Enter 4-digit OTP
   - Auto-submits when complete
   - Redirects on success
